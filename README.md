Doctors Portal - Appointments & Admin
Full portal for patients and admins: booking calendar, patient dashboards, admin management, and payments.

Frontend features:

1. Patient booking UI, dashboards, responsive views

2. Secure auth flows, appointment calendar visuals

3. Backend features

4. Auth, role-based access, appointment scheduling API

5. Admin endpoints for doctors/services, payment recording, reporting

Tech:
Next.js (frontend), Node.js, Prisma + PostgreSQL (backend)
